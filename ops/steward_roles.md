# Steward Roles

Stewards enforce the Memory Laws at runtime.

---

## Index Steward
- maintains discovery indexes
- serves lookup and query interfaces

## Decay Steward
- enforces TTL expiry
- manages confidence decay
- queues revalidation tasks

## Consensus Steward
- monitors conflicts
- enforces corroboration thresholds
- prevents unilateral elevation

## Security Steward (optional)
- monitors poisoning or abuse
- enforces access envelopes
