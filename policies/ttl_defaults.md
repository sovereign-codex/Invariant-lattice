# TTL Defaults

TTL defines how long a Meta-Hash remains valid without revalidation.

## Revalidation Triggers
- TTL expiry
- drift detection
- conflict emergence
- elevated correction rate

TTL expiry never deletes memory automatically; it disables authority until revalidated.
